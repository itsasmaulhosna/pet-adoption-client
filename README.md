🐾 PetNest – Pet Adoption Platform

## Project Name

PetNest – Full Stack Pet Adoption Platform (MERN Stack)

## Purpose

PetNest is a full-stack pet adoption platform built using the MERN stack (MongoDB, Express, React, Node.js) with authentication and secure APIs.
The platform allows users to browse pets, view details, and submit adoption requests, while pet owners can manage listings and handle adoption requests efficiently.PetNest client-side is a modern, responsive UI for a pet adoption platform where users can browse pets, view details, and send adoption requests. It connects with a secure backend API for full functionality.

## Features

- Browse and search pets with filtering system
- Detailed pet profile with adoption form
- Dashboard for pet owners (CRUD operations)
- Adoption request system with approval/rejection
- My Listings & My Requests management
- Responsive UI with dark/light mode support
- Clean UI with toast notifications (no alerts)
- Featured pets section (dynamic data)
- Search & filter pets
- Login & registration system
- Google OAuth login
- User dashboard (My Requests, Add Pet, My Listings)
- Dark/Light mode support
- Loading spinner for API calls
- Custom 404 page
- Secure authentication (Email/Password + Google OAuth)
